# 🚀 Pipeline CI/CD - Node.js + MySQL + Quality & Security

Projeto completo demonstrando uma pipeline CI/CD moderna com **Node.js, MySQL, testes automatizados, análise de qualidade e segurança**.

---

## 📋 Descrição

Este projeto evoluiu de um exemplo simples de lint para uma **API REST completa de gerenciamento de funcionários**, incluindo:

- ✅ CRUD de funcionários (Create, Update, Delete)
- ✅ Integração com banco de dados MySQL
- ✅ Testes automatizados com Jest
- ✅ Pipeline CI/CD com GitHub Actions
- ✅ Análise de qualidade com SonarCloud
- ✅ Segurança com Gitleaks e npm audit

---

## 🧰 Tecnologias

- **Node.js** - Runtime JavaScript
- **Express** - Framework para API
- **MySQL** - Banco de dados relacional
- **Jest** - Testes automatizados
- **ESLint** - Padronização de código
- **SonarCloud** - Qualidade de código
- **Gitleaks** - Detecção de secrets
- **GitHub Actions** - CI/CD
